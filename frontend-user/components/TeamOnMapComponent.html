Description

隊伍地圖元件 依照畫面類型 screenType 有兩種模式

  • 模式一: 顯示有設定據點的隊伍地圖 (screenType = 'group')
  • 模式二: 特定隊伍在地圖的呈現 (screenType = 'base')
