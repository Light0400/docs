File

projects/common/src/lib/services/scroll.service.ts

Description

提供滾動相關的服務，支援平滑滾動至元素 / 指定位置

  • 針對 iOS，因原生 scrollIntoView({behavior: 'smooth'}) 不穩定，改用自訂動畫
  • 針對其他平台，優先使用原生 scrollIntoView / scroll API
