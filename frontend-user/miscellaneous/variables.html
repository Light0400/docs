projects/common/src/lib/guards/login.guard.ts

loginGuard
Type : CanActivateFn
Default value : (route, state) => { const lus = inject(LoginUserService); const router = inject(Router); if (lus.isLogin) { return true; } return router.createUrlTree(['/', 'login'], { queryParams: { back: state.url === '/' ? undefined : state.url } }); }

登入守衛 (Login Guard)

用於路由保護，判斷使用者是否已登入。 若已登入，允許進入對應路由。 若未登入，會導向登入頁，並透過 queryParams 將原本路由作為回傳路徑。

非訪客守衛 (Not Guest Guard)

用於路由保護，判斷使用者是否已登入且非訪客帳號。 若使用者已登入且非訪客，允許進入對應路由。 若使用者未登入或為訪客，會導向登入頁，並透過 queryParams 將原本路由作為回傳路徑。

projects/common/src/lib/guards/only-web.guard.ts

onlyWebGuard
Type : CanActivateFn
Default value : (route, state) => { const platform = inject(PlatformService); const router = inject(Router); const pathPaser = inject(PathParserService); const isAppJump = route.data['AppJump']; if (platform.isWeb) { return true; } else { let url = router.createUrlTree(['/', 'home']);; if (isAppJump) { const back = route.queryParamMap.get('back'); if (back) { const slg = pathPaser.getAppOpenUrl(back); url = router.createUrlTree([slg]); } } return url; } }

只允許 Web 環境訪問的守衛 (Only Web Guard)

用於路由保護，判斷當前平台是否為 Web。

  • 若平台為 Web，允許進入對應路由。
  • 若平台非 Web：
    • 若路由 data 設定 AppJump 且 queryParam 包含 back，會將路由轉成 App 導向 URL。
    • 否則導向首頁 ('/home')。
