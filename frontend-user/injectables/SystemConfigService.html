Description

SystemConfigService

提供系統層級的設定與快取管理，包含：

  • 第三方登入參數
  • Google 地圖設定
  • 各種報表 / 活動取消 / 評論標籤的設定
  • 不雅文字庫
  • 國家縣市同步

透過快取機制減少 API 呼叫次數，並依據不同功能設置更新有效時間。
