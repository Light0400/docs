Description

Google 地圖標記封裝類別，提供自訂樣式與互動控制。

用途：

  • 建立具備自訂圖示與文字標籤的地圖標記
  • 支援「提示文字」(tip) 顯示與切換
  • 支援選取、取消選取、開關控制與焦點管理
