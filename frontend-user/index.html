TeamUp 前端程式文件

🚀 專案名稱 (Team Up)

這是一個使用 Angular + Capacitor 開發的跨平台應用程式。

Team Up 是一個單車揪團社交平台，提供單車運動愛好者參加或是規劃團體騎乘運動，除此之外也能成立在地隊伍一起經營茁壯，分享最美好的運動體驗

🛠️ 技術棧 (Tech Stack)

  • Angular - 前端框架
  • Capacitor - 原生功能橋接
  • TypeScript - 型別安全

⚙️ 開發環境需求 (Requirements)

  • [Node.js] (https://nodejs.org/zh-tw)

  • [Android Studio] (https://developer.android.com/studio?hl=zh-tw) (若需編譯 Android)

  • [Xcode] (若需編譯 iOS)

🚀 安裝與啟動 (Getting Started)

1. 安裝相依套件

npm install

2. 啟動開發伺服器 (Web)

npm run start

3. 在瀏覽器開啟

http://localhost:4200

📱 建置 (Build)

建置開發用本機 APP

npm run app-dev-build

建置發布用正式 APP

npm run app-build

同步到 Capacitor 的 android 平台

npm run app-sync-android

同步到 Capacitor 的 iOS 平台

npm run app-sync-ios

開啟 Android Studio

npx cap open android

開啟 Xcode

npx cap open ios
