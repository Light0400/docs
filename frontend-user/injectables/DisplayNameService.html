checkLangFist
checkLangFist()

檢查當前語系是否應該「名在前」。

  • true: 名在前（例如英文）。
  • false: 姓在前（例如中文、日文、韓文）。

以下語系固定為「姓在前」：

  • zh-tw, zh-cn, zh, ja, ko
Returns : boolean
  • true: 名在前（例如英文）。
  • false: 姓在前（例如中文、日文、韓文）。
displayFirst
displayFirst(firstname?: string, lastname?: string)

取得名字或姓氏的首字母（大寫）。

Example :
service.displayFirst("John", "Smith"); // "J"
service.displayFirst(undefined, "Smith"); // "S"

displayName
displayName(firstname?: string, lastname?: string)

根據名字與姓氏產生顯示名稱。

  • 若同時有名字與姓氏：
    • 語系需「名在前」→ "Firstname Lastname"
    • 語系需「姓在前」→ "Lastname Firstname"
  • 若僅有名字 → 輸出名字。
  • 若僅有姓氏 → 輸出姓氏。
  • 若皆無 → 回傳空字串。
Parameters :

格式化後的顯示名稱。
