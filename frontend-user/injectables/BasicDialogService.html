projects/common/src/lib/dialogs/basic-dialog.service.ts

Description

基礎對話框服務

提供常用對話框（錯誤提示、警示、確認、圖片瀏覽等）開啟與多國語系支援功能。 並可加入自訂動畫效果（滑入、滑出等）。
