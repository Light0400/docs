File

projects/common/src/lib/services/geolocation.service.ts

Description

定位服務

  • 提供目前位置 (getCurrentPosition)
  • 提供前景/背景定位監聽 (watchPosition)
  • 負責檢查與要求定位權限
  • 可透過 positionChange 監聽位置更新
  • 支援 Android/iOS 背景定位 (需額外設定權限)
