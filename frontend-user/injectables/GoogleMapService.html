Description

Google Map 封裝服務

  • 負責初始化 Google Map API
  • 提供地理編碼 (Geocode)、地點搜尋、靜態地圖圖片產生、路線規劃、距離/高度計算等功能

Parameters :
Name Type Optional Default value Description

lineClusterDistanceKm number No 50
