Description

驗證碼輸入控制項

  • 提供 5 個輸入框，用於輸入驗證碼 (通常為數字)
  • 支援貼上整段驗證碼、自動跳轉焦點、刪除鍵回跳、數字驗證
  • 可用於 Angular FormControl / Reactive Form
