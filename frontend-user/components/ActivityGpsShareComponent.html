Description

活動定位分享設定元件

  • 提供使用者在活動內設定是否分享運動位置
  • 可針對「分享成員」、「ˋ一般用戶:隨主辦者一同分享, 主辦/領騎: 活動開啟分享」、「個人分享開關」進行分享控制
  • 若使用者開起個人分享，但未設定分享對象，會提示前往設定
