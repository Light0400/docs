projects/common/src/lib/directives/background-image.directive.ts

Description

指令：設定元素背景圖片

功能：

  • 支援從 URL 或 API 取得圖片
  • 自動處理加載中與錯誤狀態
  • 提供事件通知圖片載入完成或發生錯誤
Example :
<div [isApiImage]="true" [appBackgroundImage]="imageUrl"></div>
