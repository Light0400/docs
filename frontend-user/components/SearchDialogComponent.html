Description

用戶查詢視窗元件

  • 提供使用者在應用內搜尋其他用戶的介面。
  • 支援推薦用戶、搜尋紀錄、即時搜尋、自動完成與清除搜尋內容等功能。
  • 整合多個子元件（推薦清單、搜尋紀錄、搜尋結果顯示等）。
