Description

確認訊息視窗元件

此元件用於顯示確認提示，例如「是否刪除」、「確定送出」等操作， 可根據傳入的 ConfirmDialogData 顯示不同樣式、文字與使用者/隊伍資訊。

Type : boolean

若傳入的樣式類型為 'top-radius'，則在外層容器套用對應的圓角樣式。

用於控制對話框的外觀（例如：自下方彈出時的上緣圓角）。
