getOrientation
getOrientation(event: DeviceOrientationEvent)

計算裝置的方向角度（北方基準，單位：度）。

  • iOS Safari 可能提供 webkitCompassHeading。
  • 透過四元數運算確保角度計算正確。
  • 若無法判斷方向則回傳 0。
Parameters :
Name Type Optional Description
event DeviceOrientationEvent No
  • 瀏覽器提供的裝置方向事件。
Returns : number

裝置方向（0~360 度，0 表示正北）。
openSetting
openSetting()

開啟當前應用程式的系統設定頁面。

Example :
deviceService.openSetting();
 - 會開啟 Android/iOS 系統設定頁面的 APP 詳細資訊
Returns : void
request
request()

請求使用裝置運動感測器的權限（iOS 13+ 必須）。

  • true: 已授權可使用裝置感測器。
  • false: 拒絕授權或裝置不支援。
Example :
const granted = await deviceService.request();
if (granted) {
  // 可以安全使用 DeviceMotionEvent
}
Returns : Promise<boolean>
  • true: 已授權可使用裝置感測器。
  • false: 拒絕授權或裝置不支援。
