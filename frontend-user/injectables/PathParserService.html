projects/common/src/lib/services/path-parser.service.ts

Description

網址路徑分析服務

提供路徑與 URL 的解析工具，用於統一處理 API 路徑、檔案路徑與應用程式內的 URL。

  • 會依照 environment 的設定與目前執行平台（Web / App）產生正確的 URL。
  • 可用於判斷 URL 是否為 API 或是否屬於本服務的應用程式 URL。
