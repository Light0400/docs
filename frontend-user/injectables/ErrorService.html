checkExistError
checkExistError(error: any, errorKeys: string[])

檢查錯誤物件中是否包含指定的驗證錯誤訊息。

  • true：錯誤訊息存在於 errorKeys 中。
  • false：不存在或錯誤物件無驗證訊息。
Example :
const exists = errorService.checkExistError(error, ['EmailRequired', 'PasswordRequired']);
Parameters :
Name Type Optional Description
error any No
  • HTTP 錯誤物件，通常包含 error.ValidationErrors。
errorKeys string[] No
  • 要檢查的錯誤訊息列表。
Returns : boolean
  • true：錯誤訊息存在於 errorKeys 中。
  • false：不存在或錯誤物件無驗證訊息。
checkShowError
checkShowError(error: any, i18nPrefix?: string)

檢查 HTTP 錯誤，若為 400 Bad Request 則透過 Dialog 顯示。

  • 若為 400 錯誤，回傳 dialog.httpError視窗關閉事件結果。
  • 其他狀態碼回傳 null。
Example :
const result = errorService.checkShowError(error, 'user');
Parameters :
Name Type Optional Description
error any No
  • HTTP 錯誤物件，通常為 HttpErrorResponse。
i18nPrefix string Yes
  • 可選的國際化前綴，用於錯誤訊息翻譯。
Returns : ReplaySubject | null
  • 若為 400 錯誤，回傳 dialog.httpError視窗關閉事件結果。
  • 其他狀態碼回傳 null。
