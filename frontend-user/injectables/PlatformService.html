projects/common/src/lib/services/platform.service.ts

Description

平台判斷服務

用於判斷目前執行環境的平台資訊（Web、App、Android、iOS）。 透過封裝 Capacitor 提供的 API，方便在應用程式中進行平台相關的邏輯判斷。

主要應用場景：

  • 根據平台載入不同的功能或 UI（如 iOS 與 Android 特殊處理）。
  • 決定是否需要走 Capacitor 的原生功能，或只使用 Web API。
