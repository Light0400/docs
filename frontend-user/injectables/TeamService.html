File

projects/common/src/lib/services/team.service.ts

Description

隊伍相關的服務

功能包含：

  • 管理隊伍成員（新增、移除、權限變更）
  • 管理稱謂（新增、修改、刪除）
  • 處理隊伍申請（同意 / 拒絕）
  • 處理退出 / 封存隊伍邏輯
  • 便利貼功能（更新、檢查是否需要顯示）
  • 圖片資源處理（Logo）

透過 Subject 發佈事件，供其他元件監聽並更新 UI。
