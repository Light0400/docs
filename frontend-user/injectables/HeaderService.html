File

projects/common/src/lib/services/header.service.ts

Description

Header 管理服務

  • 控制標題列 (顯示/隱藏、標題模板、返回按鈕、右側按鈕、Logo)
  • 管理頁面歷史紀錄，用來判斷是否可返回
  • 處理 App 返回鍵 (Android 實體返回鍵) 的行為
