Description

聊天室對話視窗畫面

  • 顯示聊天室的訊息內容
  • 提供發送訊息、附加檔案、標註使用者等功能
  • 提供聊天室設定選單，如靜音、封鎖、檢舉等功能
  • 支援搜尋聊天室訊息功能
  • 支援回覆特定訊息功能
  • 支援顯示系統訊息及使用者狀態
