ngOnDestroy
ngOnDestroy()

Angular 元件生命週期: 銷毀事件

  • 判斷是否有GPS監聽，若有必須移除監聽
  • 判斷是否有裝置方向的監聽，若有必須移除
  • 將地圖標記從地圖上清除
Returns : void
ngOnInit
ngOnInit()

Angular 元件生命週期: 初始化事件

  • 監聽 team 資料更新（包含便利貼、隊伍顏色與 Logo 變更）。
  • 等待 Google Map 載入完成後初始化 marker。
  • 根據情境建立 markerContent（隊伍、使用者或方向圖示）。
  • 若為自己位置，處理定位權限請求與方位角感應器事件。
  • 在 App resume 後重新檢查定位權限並啟動定位。
  • 最終建立 AdvancedMarkerElement 並監聽點擊事件。
Returns : void
