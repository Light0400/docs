Description

隊伍新增頁面元件

此元件負責隊伍的新增流程，包含：

  • 基本資料（名稱、公開設定）
  • 隊伍介紹
  • 圖像與背景設定
  • 標籤選擇
  • 據點位置

支援：

  • 頁面模式與 Dialog 模式開啟
  • 多步驟表單驗證
